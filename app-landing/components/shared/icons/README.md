# `components/shared/icons/`

Set SVG interne pour l'iconographie marketing de la landing.

## Role

- fournir des icones premium, sobres et coherentes pour les sections marketing
- eviter les glyphes generiques type `sparkle`, `lightbulb` ou variantes `duotone` qui donnent un rendu cheap
- garder des props simples (`size`, `className`, `title`) pour rester faciles a brancher dans les sections et pages

## Convention

- utiliser ce dossier pour les icones editoriales et de positionnement de la landing
- garder les icones utilitaires de produit ou de formulaire seulement quand elles restent neutres et lisibles
- si une nouvelle section marketing a besoin d'un picto, preferer etendre ce set SVG plutot que reintroduire un melange de librairies
