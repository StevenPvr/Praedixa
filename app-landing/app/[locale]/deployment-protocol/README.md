# Deployment Protocol

## Rôle

Ce dossier matérialise un segment de route Next.js. Les fichiers `page.tsx` définissent le rendu, le layout ou le handler HTTP de ce segment.

## Contenu immédiat

Sous-dossiers :
- Aucun élément versionné direct.

Fichiers :
- `page.tsx`

## Intégration

Ce dossier est consommé par l'application `app-landing` et s'insère dans son flux runtime, build ou test.

## Note de copy

Cette route documente le protocole de mise en place Praedixa: preuve sur historique, onboarding fixe, puis abonnement annuel.
Le contenu public de la page est alimente par `dict.pilot`; ne pas reintroduire d'ancienne cle `deployment` dans le contrat i18n.
