# Deployment

## Rôle

Ce dossier matérialise un segment de route Next.js. Les fichiers `page.tsx` définissent le rendu, le layout ou le handler HTTP de ce segment.

## Contenu immédiat

Sous-dossiers :

- Aucun élément versionné direct.

Fichiers :

- `page.tsx`

## Intégration

Ce dossier est consommé par l'application `app-landing` et s'insère dans son flux runtime, build ou test.

## Note de copy

This route is no longer a standalone public offer page.
It exists only as a permanent redirect to `contact?intent=deployment` so legacy links keep their intent without reopening a second commercial path.
