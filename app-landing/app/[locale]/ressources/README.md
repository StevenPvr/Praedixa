# Ressources

## Rôle

Ce dossier matérialise un segment de route Next.js. Les fichiers `page.tsx` définissent le rendu, le layout ou le handler HTTP de ce segment.

## Contenu immédiat

Sous-dossiers :

- `[slug]`

Fichiers :

- `page.tsx`

## Intégration

Ce dossier est consommé par l'application `app-landing` et s'insère dans son flux runtime, build ou test.

## Convention SEO

- Le hub `/ressources` doit mailler explicitement les pages piliers, les pages sectorielles, le blog et les ressources SEO a forte intention.
- Eviter d'en faire une simple page de transition faible en contenu ou en liens internes.
