# Tests

## Rôle

Ce dossier regroupe les tests associés au dossier parent `app-landing/app`. Il documente et vérifie le comportement attendu de cette zone du code.

## Contenu immédiat

Sous-dossiers :
- Aucun élément versionné direct.

Fichiers :
- `error.test.tsx`
- `global-error.test.tsx`
- `layout.test.tsx`
- `not-found.test.tsx`
- `page.test.tsx`
- `robots.test.ts`
- `sitemap.test.ts`

## Intégration

Ce dossier est consommé par l'application `app-landing` et s'insère dans son flux runtime, build ou test.
