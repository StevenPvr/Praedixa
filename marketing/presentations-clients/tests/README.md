# Tests Presentations Clients

Ce dossier centralise les tests des mini-sites clients/prospects.

## Structure

- `skolae/`
- `greekia/`
- `centaurus/`

Chaque sous-dossier contient les fichiers Vitest associes au mini-site du meme nom, tandis que la configuration runtime reste dans `marketing/presentations-clients/<nom>/vitest.config.ts`.
