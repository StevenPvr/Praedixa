# Components

## Rôle

Ce dossier regroupe les composants UI du périmètre concerné. Ils encapsulent la présentation, l'interaction et parfois une petite orchestration locale.

## Contenu immédiat

Sous-dossiers :

- `data-table`

Fichiers :

- `button.tsx`
- `card.tsx`
- `praedixa-logo.tsx`
- `skeleton.tsx`
- `stat-card.tsx`

## Intégration

Ce dossier appartient au package partagé `packages/ui` et peut être importé par plusieurs apps du monorepo.
