# Plans de Travail

Ce dossier contient les plans, cadrages et notes de conception actives ou récentes.

## Usage

- Préparer un chantier avant implémentation.
- Garder la trace des décisions de design et des hypothèses de livraison.

## Fichiers notables

- `2026-02-19-webapp-live-audit-gold-integrity-plan.md` : plan lié à l'intégrité Gold côté webapp.
- `2026-03-05-praedixa-landing-audit-refonte.md` : refonte et audit de la landing.

## Règle

- Un plan n'est pas une source de vérité runtime. Une fois implémenté, le comportement réel doit être documenté dans les README locaux et dans les docs durables du repo.
