# Access Control Policy (Template)

## Purpose

Define how Praedixa grants, reviews, and revokes access.

## Rules

- MFA required for all privileged accounts.
- Least privilege by default.
- No shared human accounts.
- Access approvals documented.
- Offboarding revocation within 24h.
- Monthly access review mandatory.

## Scope

- GitHub, cloud providers, database, CI/CD, observability tools.

## Exceptions

- Any exception must have owner, reason, and expiry date.
