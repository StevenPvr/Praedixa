# Sources Datées — Rapport Praedixa Investisseurs 2026

1. **France Travail** — Enquête BMO 2025 (publié le 11/04/2025)

   - Usage: tension recrutement, difficulté de recrutement.

2. **Dares** — Les emplois vacants (mise à jour du 16/12/2025)

3. **DREES** — Études et Résultats n°1321 (13/12/2024)
   - Lien: https://www.drees.solidarites-sante.gouv.fr/publications-communique-de-presse/etudes-et-resultats/241211_ER_Arrets-Maladie

4. **Eurostat** — E-business integration (06/02/2024)

## Règle d'usage dans le rapport

- Les chiffres externes doivent être présentés avec date de publication.
- Les chiffres non observés doivent être explicitement marqués comme hypothèses.
- Les calculs TAM/SAM/SOM et financiers restent des scénarios de travail, non des données observées.
