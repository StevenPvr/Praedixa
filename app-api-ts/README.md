# Praedixa API TS

Backend applicatif TypeScript (big-bang rewrite).

## Scope

- API produit et admin en TypeScript.
- Auth/RBAC/JWT cote API TS.
- Python reserve aux pipelines Data/ML.

## Run

```bash
pnpm --filter @praedixa/api-ts dev
```

## Build

```bash
pnpm --filter @praedixa/api-ts build
```
